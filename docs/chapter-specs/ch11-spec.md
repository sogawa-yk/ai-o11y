# 第11章 Langfuseの機能と使い方 ― 仕様書

## 章の概要

**章の目的**: Langfuseの3つの主要機能（トレーシング／評価／プロンプト管理）を理解し、基本的な使い方を実機で確認する。

**目標ページ数**: 14p

**所属する部**: 第III部 AIエージェント固有のObservability

### 11.2 機能1: トレーシング

**概要**: LLM呼び出しの入出力を並べて見る／トークンコストを集計する機能を示す。

### 11.3 機能2: 評価（Evaluation）

**概要**: トレースへのスコア付与、人手レビュー、LLM-as-judgeを紹介する。

**概要**: プロンプトのバージョン管理と、バージョン間の品質比較機能を示す。

**キーポイント**:

## 章末の理解度チェック問題

### 問題の方針
- 問題数: 4問
- 種類: 概念の確認2問、判断問題1問、設計問題1問

### 問題案
1. Langfuseの3機能それぞれを1文で説明せよ。（概念の確認）
2. OTelのトレースではなくLangfuseを使うと嬉しいのはどんな場面か。（概念の確認）
3. プロンプト変更が品質に与える影響を測定したい。Langfuseのどの機能を組み合わせて使うか。（判断問題）
4. 「LLM出力が期待と異なるケース」を検知する自動評価の仕組みを、LLM-as-judgeで設計せよ。（設計問題）
