# 用語集作成スキル

書籍企画書とコンセプトを元に、用語集を作成する。用語集は執筆時の表記統一の基準となり、レビュー時の照合にも使用される。

## 前提条件

1. `docs/book-plan.md`（書籍企画書）
2. `docs/ideas/book-concept.md`（書籍コンセプト）

## 出力先

```
docs/glossary.md
```

## テンプレート

用語集のテンプレート: `./template.md`

## 用語の分類

用語は書籍のドメインに応じて以下のカテゴリに分類する。カテゴリ名は`docs/book-plan.md`の内容に基づいて適宜決定する。

### ドメイン専門用語
書籍のトピックに固有の専門用語、パターン・概念名、プロトコル・標準名。

### プラットフォーム/サービス用語
書籍で扱う主要サービス/プロダクト名、プラットフォーム固有のサービス名・概念。

### 一般技術用語
分野横断的な技術用語、ツール・フレームワーク関連の用語。

## 品質基準

- 全ての用語に日本語表記・英語表記・定義・初出章が記載されていること
- 五十音順索引が用意されていること
- book-plan.mdの技術スタックに登場する全ての用語が含まれていること
