# 執筆ガイドラインスキル

書籍全体で統一する文体・表記・図表・引用のルールを定義する。このガイドラインは全章の執筆とレビューの基準となる。

## 前提条件

- `docs/book-plan.md`（書籍企画書）が存在すること

## 出力先

```
docs/writing-guidelines.md
```

## テンプレート

執筆ガイドラインのテンプレート: `./template.md`

テンプレートには以下のセクションが含まれる:
- 文体ルール（基本文体、文の長さ、段落の長さ、接続詞）
- 表記ルール（技術用語、数字、記号、見出し）
- 図表ルール（図表番号、キャプション、本文中の参照）
- コード例ルール（言語、形式、インデント）
- 引用・参照ルール（情報ソース、外部資料、章間参照）
- 禁止事項
